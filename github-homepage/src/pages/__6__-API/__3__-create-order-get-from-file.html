Create Order with GET_FROM_FILE

测试通过 PayPal Rest API 创建订单时，使用 experience_context.shipping_preference = "GET_FROM_FILE" 从 PayPal 账户中拉取 buyer 的 shipping address。 授权完成后会展示 order 详情中返回的 buyer 信息。

电话号码属于买家个人信息(PII), PayPal 默认不会在 payer 对象中返回。 如需收集，请在 Account Settings 的 Website Payments → Website Preferences 中开启 Contact telephone number 选项。
